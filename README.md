# CTD Regulatory Submission (Mock Project)

## Overview
This repository represents a **mock Common Technical Document (CTD) regulatory submission**
prepared in accordance with **ICH guidelines** and **US FDA / EMA regulatory expectations**.

The project is designed to demonstrate practical understanding of:
- CTD structure
- Generic drug regulatory requirements
- eCTD-aligned documentation concepts

This is an **academic and learning-based project** and does not contain real proprietary data.

---

## Regulatory Framework Followed
- ICH CTD Guidelines
- US FDA ANDA submission structure
- EMA CTD format principles

---

## CTD Module Structure

### Module 1 – Administrative Information
- Regional administrative documents
- Application and labeling overview

### Module 2 – CTD Summaries
- Quality Overall Summary (QOS)
- Nonclinical and Clinical summaries

### Module 3 – Quality (CMC)
- Drug Substance information
- Drug Product formulation
- Manufacturing and quality control overview

### Module 4 – Nonclinical (Preclinical)
- Nonclinical pharmacology and toxicology overview
- Justification for nonclinical study requirements
- Waiver-based approach for generic products

### Module 5 – Clinical
- Bioequivalence study strategy
- Pharmacokinetic parameters (Cmax, AUC, Tmax)
- Clinical waiver justification for generic drugs

- To serve as a **portfolio project for Regulatory Affairs roles**
